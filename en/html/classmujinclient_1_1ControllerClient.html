Detailed Description

Creates on MUJIN Controller instance.

Only one call can be made at a time. In order to make multiple calls simultaneously, create another instance.

Parameters
importuriThe original URI to import from. For MUJIN network files use mujin:/mypath/myfile.ext
importformatThe format of the imported file. Can be:
  • mujincollada
  • wincaps (DensoWave WINCAPS III)
  • rttoolbox (Mitsubishi RT ToolBox)
  • x (DirectX)
  • vrml
  • stl
newuriThen new URI to save the imported results. Default is to save to MUJIN COLLADA, so end with .mujin.dae . Use mujin:/mypath/myfile.mujin.dae
